# נהג בסדר — אפליקציית ניהול למורי נהיגה עצמאיים

## 📋 סקירת הפרויקט

**מטרה:** פיתוח כלי ניהול עסקי למורי נהיגה עצמאיים בישראל — פתרון לכאוס היומיומי של ניהול תלמידים, שיעורים, תשלומים ותקשורת.

**סטטוס נוכחי:** MVP עובד (HTML/CSS/JS בודד) — מוכן לבדיקת היתכנות עם מורים אמיתיים.

**קובץ עיקרי:** `nahag-baseder-v2.html`

---

## 🎯 קהל יעד

### ראשי: מורי נהיגה עצמאיים
- **כמות:** 3,500–5,500 מורים עצמאיים משלמים פוטנציאליים (מתוך ~6,000 מורים פעילים בישראל)
- **פרופיל:** גיל 30–55, עובדים 10–12 שעות ביום מהרכב, מנהלים עסק מהסמארטפון
- **כלים נוכחיים:** WhatsApp + Excel/נייר + יומן גוגל (ללא אינטגרציה)

### משני (לשלב מאוחר יותר): בתי ספר לנהיגה
- ~400 בתי ספר בישראל
- צרכים שונים: ניהול צי רכבים, ריבוי מורים, דוחות מרכזיים

## 🛠️ סטאק טכנולוגי נוכחי

```
Frontend: Vanilla HTML + CSS + JavaScript (קובץ בודד)
Storage:  localStorage (נתונים נשמרים בדפדפן בלבד)
WhatsApp: wa.me links (ללא API — פותח צ'אט עם הודעה מוכנה)
Hosting:  קובץ סטטי — אפשר לפתוח ישירות בדפדפן
```

## 🏃 Quick Start

1. פתח את `nahag-baseder-v2.html` בדפדפן
2. הוסף תלמיד חדש (עם מספר טלפון אמיתי לבדיקה)
3. קבע שיעור ולחץ "שמירה + וואטסאפ"
4. ודא שנפתח וואטסאפ עם הודעה מוכנה
5. נסה את היומן השבועי — לחץ על משבצת ריקה
